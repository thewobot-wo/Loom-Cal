<task type="auto">
  <name>Task 1: Create VoiceService and VoiceConfig</name>
  <files>LoomCal/Services/VoiceService.swift (new), LoomCal/App/VoiceConfig.swift (new)</files>
  <action>
    **VoiceConfig.swift** — static config struct (same pattern as ConvexEnv):
    - `static let apiKey = "YOUR_KEY_HERE"` (placeholder — user fills in)
    - `static let voiceId = "EXAVITQu4vr4xnSDxMaL"` (ElevenLabs "Sarah" — warm, natural female voice, good default for Loom)
    - `static let modelId = "eleven_multilingual_v2"` (latest model)

    **VoiceService.swift** — `@MainActor class VoiceService: NSObject, ObservableObject, AVAudioPlayerDelegate`:
    - `@Published var playingMessageId: String?` — currently playing message ID (nil = idle)
    - `@Published var isGenerating: Bool = false` — true while awaiting ElevenLabs API response
    - Private `audioPlayer: AVAudioPlayer?`
    - Private `audioCache: [String: Data]` — in-memory cache keyed by message ID

    **Core methods:**
    - `func play(text: String, messageId: String)`:
      1. If same messageId is paused → resume (`audioPlayer?.play()`)
      2. If different messageId is playing → stop current, start new
      3. Check `audioCache[messageId]` — if cached, play directly
      4. Otherwise: set `isGenerating = true`, call ElevenLabs API, cache result, play
      5. On iOS: configure AVAudioSession `.playback` category with `.mixWithOthers` option
         Use `#if canImport(UIKit)` guard — macOS needs no session config
    - `func pause()`: pause audioPlayer, keep playingMessageId
    - `func stop()`: stop audioPlayer, set playingMessageId = nil
    - `func togglePlayPause(text: String, messageId: String)`:
      If messageId is currently playing → pause. If paused → resume. Otherwise → play new.

    **ElevenLabs API call** (private method):
    - URL: `https://api.elevenlabs.io/v1/text-to-speech/{voiceId}`
    - Method: POST
    - Headers: `xi-api-key: {apiKey}`, `Content-Type: application/json`, `Accept: audio/mpeg`
    - Body: `{"text": text, "model_id": modelId, "voice_settings": {"stability": 0.5, "similarity_boost": 0.75}}`
    - Returns: raw audio/mpeg Data
    - Strip markdown from text before sending (remove **, *, #, `, [], () link syntax)
    - On error: set playingMessageId = nil, isGenerating = false (fail silently — button just resets)

    **AVAudioPlayerDelegate:**
    - `audioPlayerDidFinishPlaying` → set playingMessageId = nil

    Avoid: Creating multiple AVAudioPlayer instances simultaneously. Always stop/release before creating new.
    Avoid: Storing audio files on disk — in-memory cache is sufficient for chat session.

    - Below the assistant bubble content (not inside the bubble background), add a row:

      HStack(spacing: 4) {
          Button { onPlayTap?() } label: {
              if voiceService.isGenerating && voiceService.playingMessageId == nil {
                  ProgressView().controlSize(.small)
              } else {
                  Image(systemName: isThisPlaying ? "pause.circle.fill" : "play.circle.fill")
              }
          }
          .font(.caption)
          .foregroundStyle(.secondary)
      }
