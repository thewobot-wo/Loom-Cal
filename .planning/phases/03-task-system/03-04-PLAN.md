**Part A: TaskDragState and DraggableTaskRow**

Add to TodayView.swift (or as nested types):

```swift
struct TaskDragState {
    let task: LoomTask
    let location: CGPoint  // global coordinate
    var ended: Bool = false
}
```

Wrap task rows in the unscheduled section with drag capability:

```swift
// DraggableTaskRow — wraps TaskRowView with LongPress + DragGesture
// Pattern: LongPressGesture(minimumDuration: 0.2).sequenced(before: DragGesture(coordinateSpace: .global))
// — matches existing TimelineEventCard drag pattern from Phase 2
```

The draggable row must:
1. Start with `LongPressGesture(minimumDuration: 0.2)` to avoid conflicting with scroll and tap gestures
2. Sequence into `DragGesture(coordinateSpace: .global)` for position tracking
3. Show visual feedback: `.opacity(0.7)`, `.scaleEffect(1.05)` while dragging, with `.animation(.interactiveSpring())`
4. Update `@Binding var dragState: TaskDragState?` on the parent TodayView
5. On gesture end, set `dragState.ended = true`
6. On gesture end, reset offset with `.easeOut(duration: 0.15)` animation

**IMPORTANT:** The unscheduled task section sits ABOVE the ScrollView (not inside it). This is critical — DragGesture on task rows inside a ScrollView would block scrolling (Pitfall 1 from RESEARCH.md). The drag source is in the fixed-height panel above.

**Part B: Drop target coordinate resolution**

Track the timeline ScrollView ZStack's global frame using a PreferenceKey:
```swift
struct TimelineFrameKey: PreferenceKey {
    static var defaultValue: CGRect = .zero
    static func reduce(value: inout CGRect, nextValue: () -> CGRect) {
        value = nextValue()
    }
}
```

Attach to the timeline ZStack:
```swift
.background(
    GeometryReader { geo in
        Color.clear.preference(key: TimelineFrameKey.self, value: geo.frame(in: .global))
    }
)
.onPreferenceChange(TimelineFrameKey.self) { frame in
    timelineGlobalFrame = frame
}
```

When `dragState?.ended == true`, resolve the drop:
```swift
func resolveTimeSlot(globalPoint: CGPoint, timelineFrame: CGRect) -> Date? {
    let localY = globalPoint.y - timelineFrame.minY
    guard localY >= 0 && localY <= timelineFrame.height else { return nil }
    let totalHeight: CGFloat = 1440  // 24h * 60pt/h — same as DayTimelineView
    let fraction = localY / totalHeight
    let minutesFromMidnight = fraction * 24 * 60
    // Round to nearest 15 minutes
    let rounded = (minutesFromMidnight / 15).rounded() * 15
    let today = Calendar.current.startOfDay(for: calendarViewModel.selectedDate)
    return today.addingTimeInterval(rounded * 60)
}
```

On successful drop (resolveTimeSlot returns non-nil):
- Call `taskViewModel.createTimeBlock(for: task, at: resolvedDate)`
- Show brief haptic feedback (UIImpactFeedbackGenerator medium, guarded with `#if canImport(UIKit)`)

On failed drop (outside timeline frame):
- Reset drag state, no action

**Part D: Drag preview indicator**

While dragging over the timeline area, show a faint horizontal indicator line at the resolved time slot position. This gives the user feedback about where the task will land:
```swift
if let dragState, !dragState.ended {
    let indicatorY = resolveYPosition(from: dragState.location) // compute Y in timeline
    if indicatorY >= 0 {
        Rectangle()
            .fill(Color.orange.opacity(0.5))
            .frame(height: 2)
            .offset(y: indicatorY)
    }
}
```

**Part E: Undo for task completion**

Add completion undo support in TodayView. When a task is completed from the task row:
1. Call `taskViewModel.toggleComplete(task:)`
2. Set `@State var undoTask: LoomTask?` to the completed task
3. Show a floating undo banner at the bottom:
```swift
if let undoTask {
    VStack {
        Spacer()
        HStack {
            Text("Task completed")
            Spacer()
            Button("Undo") {
                Task {
                    try? await taskViewModel.toggleComplete(task: undoTask)
                }
                self.undoTask = nil
            }
        }
        .padding()
        .background(.ultraThinMaterial)
        .cornerRadius(12)
        .padding()
    }
    .transition(.move(edge: .bottom))
    .onAppear {
        Task {
            try? await Task.sleep(for: .seconds(3))
            withAnimation { self.undoTask = nil }
        }
    }
}
```
